听指令的小方块

      操作说明:
      ↑    MOV TOP n : 转上移动n格
      ↓    MOV BOT n : 转下移动n格
      ←    MOV LEF n : 转左移动n格
      →    MOV RIG n : 转右移动n格
      Shift + ↑  GO n : 向当前方向前进n格
      Shift + ↓  TUN BAC n : 向后旋转n次
      Shift + ←  TUN LEF n : 向左旋转n次
      Shift + →  TUN RIG n : 向右旋转n次
      Ctrl  + ↑  TRA LEF n : 向上平移n格，方向不变
      Ctrl  + ↓  TRA TOP n : 向下平移n格，方向不变
      Ctrl  + ←  TRA RIG n : 向左平移n格，方向不变
      Ctrl  + →  TRA BOT n : 向右平移n格，方向不变
      ~`  BUILD : 修建一个墙方格
      BRU color : 粉刷这面墙
      MOV TO col,row : 方块移动至指定坐标位置
      (使用了A*寻路算法，可避让方块寻路)
      (注: n省略默认为1/指令错误则跳过该行执行)
